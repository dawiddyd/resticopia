<p>A mobile Android application that enables efficient and straightforward data backups powered by <a href='https://restic.net/' target='_blank' rel='nofollow'>Restic</a> backup software.</p><p>The application leverages <a href='https://github.com/termux/proot' target='_blank' rel='nofollow'>termux/proot</a> technology to execute native Restic and <a href='https://rclone.org' target='_blank' rel='nofollow'>Rclone</a> Linux binaries directly on Android devices.</p><p><b>Disclaimer:</b> This is an unofficial application and is not developed or endorsed by the official Restic project team.</p><p><br><b>Key Capabilities:</b></p><ul><li>Repository Management: Create and configure Restic repositories (supports S3, B2, Rest, Local, and Rclone protocols)</li><li>Snapshot Control: Browse and manage your backup snapshots</li><li>Folder Selection: Choose which directories to include in backups</li><li>Automated Scheduling: Set up recurring backup tasks</li><li>Retention Policies: Define cleanup rules for individual folders</li><li>Live Progress: Monitor backup operations through system notifications</li><li>Rclone Integration: Access 40+ cloud storage providers via Rclone backend</li></ul>
